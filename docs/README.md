# Demo GIF

Place your demo GIF here as `demo.gif`.

The GIF should show:
1. Running `sortai test-demo` (dry-run)
2. The suggested folder structure output
3. Running `sortai test-demo --apply`
4. Confirmation prompt and file organization

**To create the GIF:**
1. Use ScreenToGif or similar tool
2. Record your terminal while running the demo
3. Save as `demo.gif` in this `docs/` folder
4. Commit and push to GitHub
5. The README will automatically display it (both on GitHub and PyPI)
